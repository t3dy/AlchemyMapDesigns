Ideas this map is trying

Style as argument
Six full visual identities — Modern Atlas, Copperplate Engraving, Illuminated Manuscript, Alchemical Noir, Woodcut & Rubric, Lapis & Gold — and every map now opens in the one that suits its material: copperplate for print-culture subjects, lapis for late antiquity, noir for hidden and rejected knowledge, woodcut for scholastic polemic. Each map's page carries a design note saying why. The reader can always switch styles live — the assignment is a curatorial argument, not a lock.

Read more: Toolkit §7 — the theme engine · Manual §4 — the six styles

Time as accumulation
The scrubber doesn't slice time, it accumulates it — press Play and events pile up through the years, so centres of gravity visibly form and migrate. On journeys the same control becomes progress-through-a-life. Under reduced-motion settings Play turns into a discrete step-through: same story, no animation.

Read more: Manual §4 — reading a data map

Transmission arcs — knowledge in motion
Arcs connect consecutive dated events of the same named person: alchemy moving as people moved. The honesty cost is stated on every map: arcs trace named figures whose itineraries survive; the anonymous carriers — merchants, manuscripts, students — vastly outnumbered them, and absence of an arc is absence of record.

Read more: Manual §8 — absence ≠ absence

Period-accurate borders
Early-modern events used to float over 21st-century borders. Now a spec can say historical_boundaries_year: 1560 and the compiler embeds approximate period polities from a locally cached, slimmed historical-basemaps dataset (snapped to the nearest cached year). It ships as a toggleable Borders layer — hover a line for the polity's name — and the map automatically gains a caveat: approximate, for orientation, never authority.
